Unified Cloud (TAUC)

Řešení Wi-Fi s cloudovou správou pro poskytovatele internetových služeb, které zjednodušuje provoz, snižuje náklady v průběhu celého životního cyklu poskytované služby a nabízí ucelené možnosti monitorování a vzdálené řešení problémů pro všechna koncová zařízení (CPE).
